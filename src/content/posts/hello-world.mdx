# Astro + Neovim + Kickstart

- add `astro`, `typescript` and `css` to your `ensure_installed` for `nvim-treesitter`
- add `astro = { 'prettierd', 'prettier', stop_after_first = true }` to your `formatters_by_ft`
- make sure Mason has installed the `astro-language-server`

# Mistakes

- make sure your `tsconfig.json` doesn't have a trailing comma
